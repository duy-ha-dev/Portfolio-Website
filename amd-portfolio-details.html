DREAM-R: Dimensionality Reduction, Feature Extraction, and Modeling for RNA Transcriptome Data

In collaboration with Baylor College of Medicine, this project aims to address the complex nature of Age-related Macular Degeneration (AMD) by utilizing machine learning to identify key genomic signatures associated with the disease. This is crucial given the multifactorial causes of AMD, which include genetic variants, environmental stress, and aging. Despite advances in treatments like Macugen and Photodynamic Therapy (PDT), a comprehensive understanding of the disease at the genetic level is still lacking.

This Python package offers a systematic approach to create machine-learning models that can classify patients into different stages of diseases. The package not only aims to provide a versatile toolset for AMD but also for genomic analysis of other complex diseases. This package includes comprehensive functionalities for data processing, feature selection, and model evaluation, thereby serving as a robust platform for genomic data analysis in various applications.

EMLT

In collaboration with Rice's Efficient and Intelligent Computing Lab (EIC), I implemented and fine-tuned a novel neuron pruning and polarization regularizer for EIC's EMLT project. I have used PyTorch and CUDA to set up my local environment to experiment with the polarization regularizer, a relatively new and promising pruning technique for machine learning. I worked on implementing the pruning technique using my own training, using the EMLT source code as a general reference for my implementation.

Website Management

In collaboration with my two friends, we worked on owning and developing our very own websites using a Virtual Private Server hosted by Vultr. We used Debian on our VPS as our main OS in order to set up our web server using nginx. After setting up additional SSH keypairs for security measures, each of us then ran our respective sites with virtual hosting, with my particular domain from Epik. This portfolio website has undergone several changes, with the latest update being 12/21/2022. During the summer of 2022, I made significant renovations to the site by utilizing wireframing techniques and the Bootstrap framework. Overhauling the website allowed me to sharpen my frontend skills as well as to give you viewers a more pleasant experience.
